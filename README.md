This repo is the Front end of the *ACL VM Scheduler* 

## Install

```shell
git clone https://github.com/dnshi/vmscheduler-ui.git && cd vmscheduler-ui
npm install
# To run dev and open http://localhost:9090
npm start
# To build prod version
npm run build
```
